SQL Sub Queries :- 

		A Subquery or inner query or Nested query is query within another SQL query
		It involves 2 select statement 

		( ----------- )
	      (	     QUERY     )
	    {			 }
	      (  {Sub Query}   )
		 (_________)

	Syntax :-
	SELECT column(s)
	FROM table_name
	WHERE col_name operator
	( subquery );
