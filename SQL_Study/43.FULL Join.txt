FULL Join :- 
     	    Returns all records when there is match in either left or right table		 (-)     (-)         
    	         								        	(|=|A(|=|)B|=|)	
		 										 (_)     (_)	
	Syntax :-
	SELECT * FROM student as s		LEFT Join
	LEFT JOIN course as c			UNION
	ON s.id = c.id 				RIGHT Join
	UNION 
	SELECT * FROM student as s
	RIGHT JOIN course as c
	ON s.id = c.id;
