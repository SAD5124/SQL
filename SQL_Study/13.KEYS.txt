KEYS :-

1} PRIMARY KEY :- (Unique, Not Null)
		It is COLUMN (or set of columns) in TABLE that UNIQUELY IDENTIFIED each ROW (Unique ID)
		There is only 1 PK & it should NOT null

2} FOREIGN KEY :-
		Foreign key is column (or set of columns) in table that refers to primary key of another table
		There can be multiple FKs
		FKs can have duplicate & null values
